Key Features

Halaqaat Management

Manage halaqaat for Baneen and Banaat with separate time slots (Subah/Asr).

Student Records

Track student information, shuba assignments, and Mumayyaz status.

Attendance Tracking

Daily attendance with present, absent, late, and excused statuses.

Exam Management

Create exams, enter marks, and finalize results with rankings.

Reports & Analytics

Detailed reports for halaqaat and students with export options.

Bilingual Support

Full Urdu (RTL) and English (LTR) language support with easy toggle.

User Roles

Admin

Full system control, manage users, halaqaat, and view all reports.

Mushrif

Ustaaz/Ustadah

Mumtahin

Setup Instructions

Requirements
  • PHP 7.4 or higher
  • MySQL 5.7 or MariaDB 10.3+
  • Apache/Nginx web server
  • PHP Extensions: mysqli, mbstring
Database Setup

// Import the database schema

mysql -u root -p < database.sql

Configuration

Edit includes/db.php with your database credentials:

define('DB_HOST', 'localhost');

define('DB_USER', 'your_username');

define('DB_PASS', 'your_password');

define('DB_NAME', 'kahaf_halaqaat');
